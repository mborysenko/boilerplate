### Introduction

Example project that uses [Dhampir Wep Application Framework
](https://github.com/dhampirjs/dhampir)

### Getting Started

#### Prerequisites

To work with the boilerplate you need:
1. Node >=20.17
2. Corepack
3. Yarn Berry

npm install -g yarn
npm install -g corepack
corepack enable

yarn install
